Accessibility Page

Accessibility statement

I want this website to be as accessible as it can be for all people and all disabilities. Using HTML 5 semantic tags and css styling I have tried to make the website easy for anyone to understand and navigate.

For speech disabilities I have given the option to send a comment and I provided an email to contact me on. Provide an address if someone doesn't have email and so they can send me a letter.

I have used HTML 5 semantics where appropriate to break up page regions
(ref:webaim.org section: 1.3.6 Identify Purpose)

Kept paragraphs short and broken up also the text contrasts the background for cognitive and visual disabilities.

All images have the alt attribute with a description or just alt=" " if it explained in text
(ref:webaim.org section: 1.1.1 Non-text Content)

The submit button, navigation items, options all get highlighted in a contrasted colour to the background when hovered or pressed. The current page in the nav bar is also highlighted for visual disabilities
(ref:webaim.org section: 1.4.11 Non-text Contrast)

When hovering over the image revealing the answer for the quiz the answer is large and obvious

Kept large text above 24 px and bold
(ref:webaim.org section: 1.4.3 Contrast (Minimum)(Level AA))
